交友π - 微信小程序前端指南

这是一个微信小程序项目，需要使用微信开发者工具来运行，而不是直接在浏览器中打开。

项目结构

前端项目结构如下：

├── app.js # 应用程序入口文件 ├── app.json # 全局配置文件 ├── app.wxss # 全局样式文件 ├── pages/ # 页面目录 │ ├── login/ # 登录页面 │ ├── index/ # 首页/匹配页面 │ ├── profile/ # 个人资料页面 │ ├── message-list/ # 消息列表页面 │ ├── chat/ # 聊天页面 │ ├── activity-list/ # 活动列表页面 │ └── ... ├── utils/ # 工具函数目录 │ └── request.js # 网络请求工具 └── project.config.json # 项目配置文件

如何运行微信小程序

步骤 1: 下载并安装微信开发者工具

从微信开发者工具官网下载并安装最新版本的微信开发者工具。

步骤 2: 导入项目

  1. 打开微信开发者工具
  2. 登录您的微信账号
  3. 点击"导入项目"按钮
  4. 填写项目信息：
    • 项目名称：交友π
    • 目录：选择项目根目录（c:\Users\35319\Desktop\交友π）
    • AppID：可以使用测试号（点击"使用测试号"）
  5. 点击"确定"按钮导入项目

步骤 3: 启动项目

导入成功后，微信开发者工具会自动编译并运行项目。您可以在模拟器中查看和测试应用。

项目功能

交友π小程序包含以下主要功能：

  • 用户登录：微信一键登录
  • 用户匹配：基于兴趣和偏好的智能匹配
  • 即时通讯：与匹配用户进行聊天
  • 活动管理：浏览、创建和参与线下活动
  • 个人资料：管理个人信息和偏好设置

后端服务

后端服务已成功启动并运行在 http://localhost:3000

测试端点：http://localhost:3000/api/users/test

注意：MongoDB数据库连接失败，这是因为本地MongoDB服务未启动。虽然应用程序会继续运行，但需要数据库的功能可能不可用。

如果需要使用完整功能，请先启动本地MongoDB服务（端口27017）。

常见问题

为什么我看不到前端界面？

这是一个微信小程序项目，必须使用微信开发者工具来运行，不能直接在浏览器中打开。请按照上述步骤安装并使用微信开发者工具。

登录失败怎么办？

确保后端服务正在运行（端口3000），然后检查微信开发者工具中的控制台输出，查看具体的错误信息。

如何修改项目配置？

可以通过修改 app.json 文件来配置页面路径、窗口样式和tabBar等。
